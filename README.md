NAME
====

Unicode::Security - A port of Perl5's Unicode Security.

SYNOPSIS
========

```perl6
use Unicode::Security;
```

DESCRIPTION
===========

Unicode::Security is a reimplementaiton of Perl5 Unicode::Security in Perl6.
